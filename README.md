# AdventOfCode 2025

This is a .NET 10 project for solving Advent of Code 2025 challenges in C# 14.0.

## Project Structure

- Each day's solution is implemented in the `AdventOfCode/2025/DayX.cs` files.
- Input files for each day are located in `AdventOfCode/2025/inputs/`.

## How to Run

1. **Clone the repository:**

2. **Open the project in Visual Studio 2026 or compatible IDE.**

3. **Select the day to run:**
- Open `Program.cs`.
- Edit the code to instantiate and run the desired day's class, e.g.:
  ```csharp
  // Example for Day 1
  new AdventOfCode._2025.Day1().RunPart1();
  // or
  new AdventOfCode._2025.Day1().RunPart2();
  ```

4. **Run the project:**
- Press `F5` or use the __Start Debugging__ command.

## Requirements

- .NET 10 SDK
- C# 14.0 support
- Visual Studio 2026 or later

## Notes

- Ensure input files for each day are present in the `AdventOfCode/2025/inputs/` directory.
- Modify `Program.cs` to select which day's solution and part to execute.
